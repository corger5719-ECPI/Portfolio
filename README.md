# My Portfolio Page

Welcome to my personal portfolio website! This page showcases my front-end development skills and projects.

## Overview

This portfolio page is a single-page application (SPA) that dynamically loads project information. It uses the browser's `sessionStorage` to store project data temporarily, preventing unnecessary reloads or API calls during the user's session.

## Languages and Libraries Used

*   **HTML5:** For the basic structure of the webpage.
*   **CSS3:** For styling and layout.
*   **JavaScript (ES6+):** For all dynamic functionality, including:
    *   DOM manipulation (`document.createElement`, `appendChild`, etc.)
    *   Web Storage API (`sessionStorage.setItem`, `sessionStorage.getItem`)
    *   JSON serialization (`JSON.stringify`, `JSON.parse`)
    *   Array methods (`forEach`)

## Dependencies

This project uses only **vanilla JavaScript** and standard browser Web APIs. There are no external libraries or frameworks required to run the page.

## Running the Page

1.  Clone the repository to your local machine.
2.  Open the `index.html` file in your preferred web browser.

The JavaScript will automatically handle storing and displaying the project information upon loading.
